# GGbot
GGbotは[Discord.js](https://github.com/discordjs/discordjs)によって書かれた，"ゲーム同行会"向けのDiscord Botです。
# 機能
| コマンド | 引数 | 説明 |
| :--- | :--- | :--- |
| `/ping` | なし | "pong!"と返します |
| `/echo` | 文字列(必須) | 入力した文字列を返します |
| `/role` | ロール(任意) | コマンドを入力した人に[から]ロールを追加[削除]します<br>ロールが入力されていないときは，ロール一覧を表示します |
